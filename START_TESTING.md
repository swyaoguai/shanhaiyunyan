## ⚠️ 如果遇到问题

### 问题: 程序无法启动

**检查清单**:
- [ ] 是否在桌面或文档文件夹（不要在 C:\Program Files）
- [ ] 是否配置了 .env 文件
- [ ] 查看 startup_error.txt 文件

### 问题: 配置验证失败

**解决方法**:
```
1. 检查 .env 文件是否存在
2. 检查 API Key 是否正确
3. 查看控制台错误信息
```

### 问题: 端口被占用

**解决方法**:
```
程序会自动尝试端口 5657, 5658...
查看控制台显示的实际端口号
```

### 问题: 浏览器没有自动打开

**解决方法**:
```
手动打开浏览器访问:
http://localhost:5656
```
