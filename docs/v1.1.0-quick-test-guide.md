## 常见问题排查

### 问题 1: 程序无法启动

**检查**:
- [ ] 是否在受保护目录（C:\Program Files）
- [ ] 是否有写入权限
- [ ] 查看 `startup_error.txt`

**解决**:
- 移动到桌面或文档文件夹
- 以管理员身份运行

### 问题 2: 配置验证失败

**检查**:
- [ ] `.env` 文件是否存在
- [ ] API Key 是否正确
- [ ] 查看控制台错误信息

**解决**:
- 检查 `.env` 配置
- 填入正确的 API Key

### 问题 3: 端口被占用

**检查**:
- [ ] 是否已有实例在运行
- [ ] 其他程序是否占用端口

**解决**:
- 关闭其他实例
- 程序会自动尝试其他端口

### 问题 4: 页面无法加载

**检查**:
- [ ] 浏览器是否正确打开
- [ ] 端口号是否正确
- [ ] 查看控制台日志

**解决**:
- 手动访问 http://localhost:5656
- 检查控制台显示的实际端口

### 问题 5: API 调用失败

**检查**:
- [ ] API Key 是否有效
- [ ] 网络连接是否正常
- [ ] API Base URL 是否正确

**解决**:
- 验证 API Key
- 检查网络连接
- 查看错误日志
