## 1. 项目概述

### 1.1 项目简介
这是一个基于多Agent协作的AI小说创作系统，采用 Coordinator-Worker 架构模式，提供完整的小说创作工作流。

### 1.2 技术栈
- **后端**: Python 3.10+, FastAPI, Pydantic
- **前端**: 原生 JavaScript (模块化), CSS3
- **数据库**: SQLite (全文检索), ChromaDB (向量存储)
- **LLM**: OpenAI 兼容 API

### 1.3 核心功能模块
- 多Agent协作系统 (世界观/大纲/章节/润色/评估)
- 知识库系统 (三层架构)
- Copilot AI 助手
- 无限续写功能
- Token 统计

```python
# 将项目元数据迁移到 SQLite
class ProjectStore:
    def __init__(self, db_path: str):
        self.conn = sqlite3.connect(db_path)
        self._init_tables()

    def _init_tables(self):
        self.conn.execute('''
            CREATE TABLE IF NOT EXISTS projects (
                id TEXT PRIMARY KEY,
                name TEXT NOT NULL,
                description TEXT,
                created_at TEXT,
                updated_at TEXT,
                word_count INTEGER DEFAULT 0,
                chapter_count INTEGER DEFAULT 0
            )
        ''')

    def create_project(self, name: str, description: str = "") -> Project:
        with self.conn:
            # 使用事务
            project_id = str(uuid.uuid4())[:8]
            self.conn.execute(
                'INSERT INTO projects VALUES (?, ?, ?, ?, ?, ?, ?)',
                (project_id, name, description, now, now, 0, 0)
            )
        return Project(id=project_id, name=name, description=description)
```

```python
class LLMResponseCache:
```

```python
# 添加HTTPS中间件
class HTTPSRedirectMiddleware:
    def __init__(self, app, enabled: bool = True):
        self.app = app
        self.enabled = enabled

    async def __call__(self, scope, receive, send):
        if self.enabled and scope["type"] == "http":
            headers = dict(scope.get("headers", []))
            if headers.get(b"x-forwarded-proto", b"http") != b"https":
                await send({
                    "type": "http.response.redirect",
                    "status": 301,
                    "headers": [(b"location", b"https://" + headers[b"host"] + scope["path"])],
                })
                return
        await self.app(scope, receive, send)
```

```python
class BatchEmbeddingService:
    def __init__(self, batch_size: int = 32):
        self.batch_size = batch_size

    async def embed_batch(self, texts: List[str]) -> List[List[float]]:
        results = []
        for i in range(0, len(texts), self.batch_size):

            results.extend(embeddings)
```

```javascript
// 对于长列表使用虚拟滚动
class VirtualList {
  constructor(container, itemHeight, renderItem) {
    this.container = container;
    this.itemHeight = itemHeight;
    this.renderItem = renderItem;
    this.visibleItems = Math.ceil(container.clientHeight / itemHeight) + 2;
  }

  render(items, scrollTop) {
    const startIdx = Math.floor(scrollTop / this.itemHeight);
    const endIdx = Math.min(startIdx + this.visibleItems, items.length);

    // 只渲染可见项
    this.container.innerHTML = items.slice(startIdx, endIdx)
      .map((item, i) => this.renderItem(item, startIdx + i))
      .join('');
  }
}
```
