## 使用说明

### 环境变量配置
创建 `.env` 文件：
```env
# LLM配置
LLM_API_KEY=your-api-key
LLM_API_BASE=https://api.openai.com/v1
LLM_MODEL=gpt-4

# 服务器配置
SERVER_PORT=5656

# 频率限制
RATE_LIMIT_REQUESTS_PER_MINUTE=60

# 日志
LOG_LEVEL=INFO
LOG_SANITIZE_SECRETS=true
```

### 运行测试
```bash
pytest novel_agent/tests/test_core_business.py -v
```
