## 1. 项目简介

### 1.1 什么是小说创作Agent智能体？

这是一个**基于多Agent协作的AI小说创作系统**，采用 Coordinator-Worker 架构模式，通过智能路由和专业分工，帮助你从零开始创作一部完整的长篇小说。

### 1.2 核心特点

| 特点 | 说明 |
|------|------|
| 🤖 **多Agent协作** | 10+个专业Agent分工协作，各司其职 |
| 🧠 **智能路由系统** | Router Agent自动识别意图，调用合适的Agent |
| 📖 **知识库系统** | 三层架构设计，支持混合检索（向量+全文） |
| 🎯 **灵活流程** | 不强制固定流程，用户可随时跳过、重复、打断 |
| 📊 **透明化执行** | 每个操作都会明确告知用户当前状态和下一步 |
| 🔒 **质量保障** | 多重质量检测：禁用词检测、AI降智检测、一致性检查 |
| 💾 **多项目管理** | 支持多个项目独立管理，数据隔离 |
| 📤 **多格式导出** | 支持TXT、Markdown、HTML、EPUB格式导出 |

### 1.3 适用场景

✅ 网络小说创作（玄幻、都市、科幻、武侠等）  
✅ 长篇小说规划与撰写  
✅ 世界观设定构建  
✅ 章节内容生成与润色  
✅ 剧情大纲规划  
✅ 角色设定管理  
✅ 创作灵感辅助

## 6. 快速上手

### 6.1 5分钟创建第一章

#### 步骤1：创建项目
1. 打开浏览器访问 `http://localhost:5656`
2. 点击左上角"项目选择器"
3. 点击"新建项目"
4. 输入项目名称（如：《修仙逆袭传》）
5. 点击"创建"

#### 步骤2：与AI对话
在右侧Copilot助手中输入：
```
我想写一部玄幻修仙小说
```

系统会自动调用**Communicator Agent**收集需求。

#### 步骤3：回答AI的问题
根据AI的提问，提供以下信息：
- 主题风格：热血修仙，主角逆袭成长
- 主角设定：天赋平平但意志坚定的少年
- 剧情构思：从小门派弟子成长为修仙界巨擘
- 篇幅规划：3卷，每卷30章

#### 步骤4：自动构建世界观
系统会自动调用**Worldbuilder Agent**构建世界观，包括：
- 力量体系（炼气→筑基→金丹→元婴→化神）
- 地理环境（东域、西域、中州）
- 势力派系（天剑宗、魔煞教等）

#### 步骤5：生成故事大纲
系统自动调用**Outliner Agent**规划3卷90章的完整大纲。

#### 步骤6：创作第一章
在Copilot中输入：
```
开始写第一章
```

系统会自动：
1. 制定章纲
2. 读取上下文
3. 创作正文（2000-3000字）
4. 质量检测
5. 保存文件

完成后，你会在左侧章节列表看到：
```
📖 第1章-初入仙门-2156字.md
```

## 7. 功能详解

### 7.1 项目管理

#### 创建项目
1. 点击左上角"项目选择器"
2. 点击"新建项目"
3. 输入项目名称
4. 选择项目类型（可选）
5. 点击"创建"

#### 切换项目
1. 点击"项目选择器"
2. 从列表中选择要切换的项目
3. 系统会自动加载该项目的所有数据

#### 导入项目
1. 点击"导入项目"
2. 选择项目文件夹或ZIP包
3. 系统会自动识别并导入

#### 导出项目
支持多种格式导出：
- **TXT格式**：纯文本，兼容性最好
- **Markdown格式**：保留格式，支持章节标题
- **HTML格式**：网页格式，支持样式
- **EPUB格式**：电子书格式，支持目录

### 7.2 世界观构建

#### 自动生成世界观
1. 在Copilot中输入："帮我构建世界观"
2. 回答AI的问题（小说类型、主题等）
3. 系统自动生成完整世界观

#### 世界观包含内容
- **力量体系**：修炼境界、能力划分、突破条件
- **地理环境**：世界地图、地域特色、气候环境
- **历史文化**：历史背景、文化传统、重大事件
- **势力派系**：门派组织、势力关系、权力结构
- **特殊设定**：独特规则、禁忌、传说

#### 手动编辑世界观
1. 点击左侧"世界设定"
2. 选择要编辑的部分
3. 直接修改内容
4. 点击"保存"

### 7.3 角色管理

#### 创建角色
1. 点击左侧"角色档案"
2. 点击"新建角色"
3. 填写角色信息：
   - 姓名
   - 性格特点
   - 背景故事
   - 能力设定
   - 目标动机
4. 点击"保存"

#### 角色关系网络
系统会自动构建角色间的关系图谱：
- 亲属关系
- 师徒关系
- 朋友/敌人关系
- 情感关系

#### 快速引用角色
在Copilot中使用 `@角色名` 快速引用角色设定。

### 7.4 大纲规划

#### 创建故事大纲
1. 在Copilot中输入："帮我规划大纲"
2. 系统会基于世界观和角色生成大纲
3. 包含：起承转合、分卷规划、章节安排

#### 细化章节大纲
1. 选择要细化的卷
2. 系统会为每章生成详细章纲
3. 包含：核心事件、剧情目标、角色变化、伏笔钩子

#### 手动调整大纲
1. 点击左侧"大纲"
2. 选择要修改的章节
3. 编辑章纲内容
4. 点击"保存"

### 7.5 章节创作

#### AI自动撰写
1. 在Copilot中输入："写第X章"
2. 系统会自动：
   - 制定章纲
   - 读取上下文
   - 创作正文
   - 质量检测
   - 保存文件

#### 手动撰写
1. 点击"新建章节"
2. 在编辑器中自由创作
3. 可随时使用"AI续写"功能
4. 点击"保存"

#### 无限续写
1. 选择要续写的章节
2. 点击"AI续写"按钮
3. 系统会基于已有内容智能续写
4. 可多次续写，直到满意

### 7.6 内容优化

#### 智能润色
1. 选中要润色的文字
2. 点击"润色"按钮
3. 系统会优化文字表达，增加细节
4. 预览润色结果
5. 点击"应用"

#### 字数扩写
当章节字数不足2000字时：
1. 系统会自动提示
2. 点击"扩写"按钮
3. 系统会自然扩充内容
4. 保持原文风格一致

#### 质量检测
系统会自动检测：
- ✅ 字数是否达标
- ✅ 是否有禁用词
- ✅ 是否有AI降智特征
- ✅ 逻辑是否连贯

### 7.7 Copilot助手

#### AI对话
随时与AI助手对话，获取创作建议：
```
用户："这个情节怎么展开比较好？"
Copilot："建议从以下几个角度展开..."
```

#### @引用功能
- `@角色名`：引用角色设定
- `@章节名`：引用章节内容
- `@世界观`：引用世界观设定
- `@大纲`：引用故事大纲

#### 快速操作
- 生成角色对话
- 描写场景
- 设计情节转折
- 创建伏笔
- 设计冲突

## 8. Agent系统详解

### 8.1 Router Agent（智能路由器）⭐核心

**这是整个系统最重要的Agent！**

#### 职责
- 分析用户输入，识别意图
- 决定调用哪个Agent或Skill
- 检查执行条件是否满足
- 透明化告知用户决策过程

#### 意图识别表

| 用户输入示例 | 识别意图 | 调用对象 |
|------------|---------|----------|
| "我想写小说" | 需求收集 | Communicator Agent |
| "帮我设计世界观" | 构建世界观 | Worldbuilder Agent |
| "创建主角" | 角色设定 | CharacterBuilder Agent |
| "规划大纲" | 大纲规划 | Outliner Agent |
| "写第5章" | 章节创作 | CreativeWriter Agent |
| "继续写" | 续写内容 | ContinuousWriter Agent |
| "润色这段" | 内容润色 | Polisher Agent |
| "检查质量" | 质量检测 | QualityValidator Agent |
| "搜索修仙设定" | 网络搜索 | web_search Skill |
| "最近有什么热梗" | 热点搜索 | trends_search Skill |
| "主角什么境界" | 知识库检索 | knowledge_base Skill |

### 8.2 Communicator（沟通助手）

#### 职责
- 与用户多轮对话
- 收集创作需求
- 引导用户提供必要信息
- 输出结构化需求JSON

#### 智能行为
- ✅ 不重复询问已知信息
- ✅ 根据回答逐步深入
- ✅ 识别隐含需求
- ✅ 提供具体建议

### 8.3 Worldbuilder（世界观构建器）

#### 职责
- 基于需求生成世界观
- 设计力量体系
- 规划地理环境
- 构建势力关系

### 8.4 CreativeWriter（创意写作师）⭐核心

#### 职责
- 基于章纲创作正文
- 实时统计字数
- 遵守三重约束
- 自动检测质量问题

#### 三重约束
1. **禁用词汇表**：避免使用禁用词
2. **反降智规则**：避免AI写作特征
3. **知识库设定**：遵守已有设定

### 8.5 QualityValidator（质量验证师）

#### 12种降智特征检测
1. 逗号滥用
2. 转折词开头过多
3. 句式重复
4. 省略号滥用
5. 模糊词过度
6. 机械段落结构
7. 对称句式过度
8. 概括总结句过多
9. 机械情绪描写
10. 四字词堆砌
11. AI高频词汇
12. 模型降智

### 8.6 其他Agent

- **Evaluator**：质量评估器
- **Polisher**：内容润色器
- **ContentExpansion**：内容扩展师
- **ContinuousWriter**：无限续写
- **Outliner**：大纲生成器

## 12. 故障排除

### 12.1 启动失败

**问题**：运行`python run.py`后报错

**解决方案**：
1. 检查Python版本：`python --version`
2. 重新安装依赖：`pip install -r requirements.txt --force-reinstall`
3. 检查端口占用：更换端口或关闭占用程序

### 12.2 API调用超时

**问题**：API调用一直超时

**解决方案**：
1. 检查网络连接
2. 尝试更换API地址
3. 增加超时时间（在设置中）

### 12.3 数据丢失

**问题**：项目数据丢失

**解决方案**：
1. 检查`data/projects/`目录
2. 查看备份文件
3. 使用导入功能恢复

### 12.4 内存不足

**问题**：系统提示内存不足

**解决方案**：
1. 关闭其他程序
2. 减少同时打开的章节数
3. 清理知识库缓存
